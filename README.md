# 📱 Projeto Flutter

App desenvolvido em **Flutter** como projeto prático, com foco em estrutura de telas, navegação e uso de widgets fundamentais.

---

## 🚀 Sobre o projeto

Este é um **projeto em Flutter** que demonstra o uso de telas, navegação entre rotas, widgets básicos e lógica de interação com o usuário.  
Ele foi criado com o intuito de aplicar os conhecimentos de Flutter em um app funcional e organizado.

---

## 🧩 Funcionalidades

- 💡 Estrutura de telas com navegação entre páginas
- 🧠 Uso de widgets essenciais do Flutter
- ✨ Layout responsivo para diferentes tamanhos de tela
- 📍 Organização de pastas por funcionalidade
- 📱 Aplicativo pronto para rodar em Android e iOS

## 📥 Como executar

Siga os passos abaixo para rodar o app:

1. Clone o repositório:
```bash
git clone https://github.com/caiodantass/Projeto_Flutter.git
```

2. Entre na pasta do projeto:
```bash
cd Projeto_Flutter
```

3. Instale as dependências:
```bash
flutter pub get
```

4. Conecte um dispositivo (ou rode em emulador):

5. Execute o app:
```bash
flutter run
```

## 🛠️ Pré-requisitos

Antes de rodar o projeto, certifique-se de ter:

✔️ Flutter SDK instalado
✔️ Editor de código (VS Code, Android Studio, etc.)
✔️ Emulador ou dispositivo conectado
